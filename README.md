# 🚀 Trading Bot Kotak — Breakout Logic System

A professional, fully automated algorithmic trading system for Kotak Neo. This bot features high-frequency breakout strategies, a real-time React dashboard, and a secure multi-user login system.

## 🎯 Key Features
*   **Automated Breakout Strategy**: Advanced Supertrend-based logic.
*   **Live Dashboard**: Monitor trades, P&L, and WebSocket status in real-time.
*   **Secure Session Logging**: Track every trading session with Kotak ID and Client ID separation.
*   **Multi-User Support**: Cloud-synced registration and user management.
*   **Automated DB Viewer**: Built-in viewer for Signup Data and User Sessions.

## 🛠️ Tech Stack
*   **Frontend**: React.js / Vite
*   **Backend**: FastAPI (Python)
*   **Database**: PostgreSQL & SQLite
*   **Broker API**: Kotak Neo (HSM/WebSocket)
